Este site foi construído com base em um modelo do figma do endereço
https://www.figma.com/file/bUGIPys15E78w9bs1l4tgS/HooBank

Utiliza bootstrap para a construção do leiaute do site. O projeto é
estático e tem como propósito o estudo do uso do bootstrap para
construção de leiautes rápidos e responsivos, usando componentes
já prontos do bootstrap, como a navbar, e importando ícones através
de APIs.
